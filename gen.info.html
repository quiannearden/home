Memory

RAM

RAM stands for Random Access Memory. It is the type of memory that can be accessed randomly without the preceding bytes.RAM has a high storage capacity which ranges from 64 megabytes to 16 gigabytes. It is a volatile memory that stores the data until the computer is on once the computer is turned off the data that has been stored will be erased. It is a read-write memory of the computer. RAM falls into the category of storage.

ROM

ROM stands for, Read-Only Memory. It is a memory containing hardwired instructions that the computer uses when it boots up, before the system software loads. Just like RAM the Random Access Memory (ROM) also has the data storage capacity however it is a non-volatile memory which means that even if the computer power is turned off the data will be stored permanently when the computer is turned on. The data in ROM can only be read by the CPU (Central Processing Unit). When compared to RAM it has smaller storage capacity and it is slower.

CACHE

The cache is a program that helps speed up the working system of a computer. When the computer is being used by a user, the cache memory stores the information program that have been used frequently, for example, if I have used the website Code Hs frequently be going back to it by deleting the tab and visiting again or just moving from one tab to the other this cache memory is storing the data while the process is going on with the user and the computer. When accessing the instruction quickly it increases the speed of the software program on the computer. Cache falls into the group of storage which temporarily stores the data and that can be fetched easily.
